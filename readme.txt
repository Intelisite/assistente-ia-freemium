Assistente virtual com inteligência artificial personalizável. Ideal para sites de qualquer nicho.

== Description ==

O **Assistente IA** é um plugin que adiciona um chat flutuante ao seu site WordPress, com integração à API da OpenAI.  
O visitante pode interagir com um assistente virtual empático, acolhedor e inteligente.

Desenvolvido por **[Intelisite.io](https://intelisite.io)**, esse plugin permite:

- Inserir sua própria chave da OpenAI
- Personalizar o comportamento do assistente com base no nicho do seu site
- Fornecer instruções personalizadas para a IA
- Customizar as cores do botão, fundo e texto do chat
- Versão Freemium com limite de mensagens por visitante
- Sugestão de upgrade automático após o limite

Ideal para blogs, sites de psicologia, pets, educação, atendimento ao cliente, e muito mais.

== Instalação ==

1. Faça o upload da pasta `assistente-ia` para a pasta `/wp-content/plugins/` ou instale via painel WordPress (Plugins > Adicionar novo > Enviar plugin).
2. Ative o plugin no menu **Plugins**.
3. Acesse o menu **Assistente IA** no painel e configure sua chave da OpenAI, cores, nicho e instruções.
4. Pronto! O chat estará disponível no seu site.

== FAQ ==

= Preciso ter uma conta na OpenAI? =  
Sim. Você deve criar uma conta em https://platform.openai.com/ e gerar sua própria chave de API.

= O plugin é gratuito? =  
Sim, esta versão é gratuita. Pode haver uma versão Pro com recursos extras.

= O plugin funciona com qualquer tema? =  
Sim, ele injeta o código diretamente no rodapé do site (`wp_footer`), compatível com qualquer tema moderno.
